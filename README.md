
## Rodando localmente

Clone o projeto

```bash
  $ git clone https://github.com/murilo-alvesmelo/teste-front-end.git
```

Entre no diretório do projeto

```bash
  $ cd teste-front-end
```

Instale as dependências

```bash
  $ npm install
```

Inicie a aplicação

```bash
  $ npm start
```
